## Overview
Flashcard Quiz Generator is an Android application that helps users convert learning content into **interactive flashcard-based quizzes**.  
Users can manually enter questions and answers or select files (PDF / Image upload supported) and practice through a simple quiz interface with difficulty levels.

This project focuses on **core Android development concepts** and clean, intuitive UI design.
